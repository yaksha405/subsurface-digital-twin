# 地下裂缝探测机器人集群数字孪生平台 v2 — 设计规范

## 一、产品定位

异构小型特种机器人集群（200台）撒入地下裂缝带，仿生蠕动式探测，传感器数据回传。
用户：煤矿/金矿/油气行业工程师、算法开发者、测试人员。

**核心价值**：把地下看不见的裂缝网络和参数数据变成可交互的3D可视化平台。

## 二、P0 — DeepSeek 大模型真实接入

### 2.1 Settings 面板
- 右上角齿轮图标 → Dialog
- 字段：Provider（DeepSeek/OpenAI/Qwen/自定义）、Base URL、API Key、Model Name
- 保存到 localStorage，运行时可切换
- Provider 预设默认值：
  - DeepSeek: `https://api.deepseek.com` / `deepseek-chat`
  - OpenAI: `https://api.openai.com/v1` / `gpt-4o`
  - Qwen: `https://dashscope.aliyuncs.com/compatible-mode/v1` / `qwen-plus`

### 2.2 System Prompt（地质裂缝分析专家）
```
你是地下裂缝探测机器人集群数字孪生平台的AI助手。
专业领域：地质工程、岩土力学、地下空间探测、煤矿/金矿/油气开采。

职责：
- 分析裂缝网络中的传感器数据
- 提供地质稳定性评估和灾害预警建议
- 解释裂缝参数的工程意义
- 辅助工程师做决策

当前场景传感器数据和选中区域参数会作为上下文提供。
回答时使用中文，引用具体数值，给出专业建议。
```

### 2.3 场景数据注入
每次对话自动携带：
- 当前场景类型（煤矿/金矿/油气）
- 选中区域的传感器数据聚合
- 选中裂缝的参数
- 机器人集群状态摘要

## 三、P0 — 3D 场景重构

### 3.1 从隧道 → 岩体裂缝网络

**视觉呈现**：
- 半透明岩体块（约 200m×100m×80m 的地质体）
- 内部嵌入多条蜿蜒曲折的裂缝（管状/带状几何体）
- 裂缝形成网络：主裂缝 + 分支裂缝 + 交叉点
- 机器人用小发光点表示，沿裂缝路径分布
- 裂缝可根据参数着色（应力/瓦斯/温度热力图）
- 岩体可旋转、可做剖面切割查看内部

### 3.2 裂缝数据模型（基于论文真实参数）

| 参数 | 范围 | 单位 | 来源 |

## 四、P1 — 行业场景切换

### 三种场景

| 场景 | 传感器侧重 | 核心实验 |
|------|-----------|----------|
| 煤矿 | CH4/CO/H2S/温度/应力/水压/微震/声发射 | 瓦斯扩散、顶板稳定性、突水预警、岩爆预测 |
| 金矿 | 应力/位移/微震/声发射/岩体强度/渗透率 | 岩爆预警、围岩稳定性、应力集中区 |
| 油气 | 孔隙压力/渗透率/孔隙度/裂缝开度/流体化学/地应力 | 连通性分析、渗透率评估、水力压裂效果 |

## 五、P1 — 标注与测量工具（参考 Potree + CloudCompare）

### 5.1 设计原则（参考 Potree 测量工具）
- **模式切换**：工具栏按钮激活工具，再次点击或 ESC 取消
- **光标变化**：激活工具时改变光标样式
- **防误操作**：需要 click 确认，右键取消当前操作
- **实时预览**：鼠标移动时实时显示预览线/区域
- **结果持久化**：标注结果可保存/删除

### 5.2 四种工具

| 工具 | 交互 | 产出 |
|------|------|------|
| 剖面线 | 点2个点 → 生成线 | 折线图（应力/瓦斯/温度沿路径变化） |
| 区域框选 | 拖拽矩形 | 聚合统计面板 |
| 文字标注 | 点击+输入 | 标注气泡（参考 Potree Annotation） |
| 测距 | 点2个点 | 3D 距离标注（IKEA 风格） |

### 5.3 剖面分析面板
选中剖面线后，右侧弹出折线图面板：
- X轴：沿线的距离
- Y轴：可选参数（应力/瓦斯/温度/渗透率等）
- 多条曲线叠加对比

## 六、P2 — 虚拟实验面板

选中区域后，右侧弹出实验选择：
- 列出可用实验（根据场景类型过滤）
- 点击执行 → 基于传感器数据计算
- 结果以图表/数值/热力图展示
- AI 给出专业解读
