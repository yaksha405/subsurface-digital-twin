# 全量 UX 审计报告 — 多角色用户视角

> 审计日期: 2026-06-13
> 审计角色: 矿场老板(矿长) / 地下检测所测绘工程师 / 核电站检修工人
> 审计范围: 全部按钮、面板、交互、3D 场景、导出功能

---

## 一、阻断性问题 (CRITICAL — 必须修，否则产品不可用)

### C1. 告警无法"确认/消除" — 只能看不能操作
- **现象**: `AlertFeed` 告警有 `acknowledged` 字段，UI 上已确认的告警显示半透明，但**没有任何按钮**能让用户确认或消除告警
- **影响**: 矿长看到一个红色瓦斯告警，想做"已知晓/已派人处理"标记，无处可点。告警列表越积越多，最终变成噪音
- **建议**: 每条告警右侧加"确认"按钮（✓ 图标），点击后标记 acknowledged；顶部加"全部确认"按钮

### C2. AI 对话发送失败时无限卡死
- **现象**: `ChatPanel.handleSend()` 无 try/catch。如果 API Key 无效、网络断开、API 限流，`streamChat` 抛异常后 `streamingText` 永远停留在空字符串，光标无限闪烁
- **影响**: 用户输入问题后看到永久 loading，不知道失败了，无法继续对话
- **建议**: 加 try/catch/finally，catch 中添加错误消息气泡，finally 中清除 streamingText

### C3. 测量工具激活后无操作指引
- **现象**: 点击 F1(剖面线)/F2(区域框选)/F3(文字标注)/F4(测距)后，只显示"按 ESC 取消"，但**不告诉用户接下来该做什么**
- **影响**: 矿长想量一段裂缝的长度，激活"测距"后面对 3D 场景一脸懵，不知道要点击两个点
- **建议**: 激活工具后在 3D 场景顶部浮出操作提示条："请点击两个点测量距离 · 右键取消"

### C4. 无移动端/平板适配 — 布局固定三栏
- **现象**: 左栏 `min-w-[240px]`、右栏 `min-w-[260px]`，3D 场景用 WebGL。在 iPad/手机上完全无法使用
- **影响**: 矿长在井下用平板巡检时打不开系统；投资人用手机演示时直接黑屏
- **建议**: 至少做 iPad 横屏适配（面板可折叠/抽屉式）；手机端提供只读安全看板（仅显示告警+评分+趋势）

### C5. 无声音/弹窗告警通知
- **现象**: 新 danger 级告警进入列表时仅在左侧面板闪烁，无声音、无全屏弹窗、无浏览器通知
- **影响**: 矿长正在看机器人列表或 AI 对话时，一条瓦斯爆炸预警悄悄滑入告警列表底部，完全被忽略
- **建议**: danger 级告警触发时：(1) 播放告警音 (2) 屏幕顶部弹出红色横幅 (3) 支持 Web Notification API

## 二、重大体验缺陷 (MAJOR — 严重影响使用效率)

### M1. 左侧面板信息过载 — 关键数据被埋没
- **现象**: 控制台从上到下塞了 7 个模块: 系统状态 > 告警 > 机器人(搜索+3个筛选+列表) > 趋势 > 图层 > 阈值 > 工具。在 1080p 屏幕上需要滚动 3-4 屏
- **影响**: 矿长最关心的"当前最危险的告警在哪"被压在系统状态下面；瓦斯阈值滑块在很底部
- **建议**: (1) 告警模块默认置顶且 danger 级自动展开 (2) 支持拖拽排序面板模块 (3) 机器人列表默认折叠

### M2. 右侧空状态提示不适配场景
- **现象**: 未选中裂缝时显示"点击 3D 场景中的**裂缝**查看详细参数"。但在管线/核反应堆/炼油厂场景下不叫"裂缝"
- **影响**: 核电站工人看到"点击裂缝"会困惑——反应堆里哪来的裂缝？
- **建议**: 根据场景动态文案: 管线→"点击管道查看检测数据"、核反应堆→"点击管道查看检修参数"

### M3. 指南针不随相机旋转 — 方向误导
- **现象**: 左上角指南针的 N/S 标记是静态的，3D 场景旋转时指针不动
- **影响**: 测绘工程师旋转视角后，指南针指北与实际场景朝向不符，导致方位判断错误
- **建议**: 用 `useFrame` 读取相机 azimuth 角度，旋转指南针 SVG

### M4. 开发者标签暴露给终端用户 — 不专业
- **现象**: 3D 场景底部有 "Three.js R3F" "Potree LOD" "Open3D Backend" "DeepSeek AI"；控制台标题旁有 "SHADCN · RADIX UI"
- **影响**: 煤矿采购方看到这些技术栈标签会觉得是个 demo 而非成品，降低信任度
- **建议**: 移到 Settings → About 页面；或改成客户感知的标签如"激光雷达点云引擎""AI 智能分析引擎"

### M5. ExportHub 无 Escape 关闭、无真实进度条
- **现象**: (1) 自定义 Modal 未处理 Escape 键 (2) LAS 导出 ~60000 点时只显示转圈+文字，无百分比进度
- **影响**: 测绘工程师点导出 LAS 后等了 5 秒以为卡死了，反复点击
- **建议**: (1) 加 `onKeyDown` Escape 闭合 (2) 导出过程分阶段显示: "生成体素点(45%)→写入二进制头(80%)→完成"

### M6. 无导出文件命名/批量导出
- **现象**: 导出文件用系统生成的文件名，且一次只能导出一种格式
- **影响**: 矿长每周要给安监局交全套资料（PDF+CSV），要点 4 次导出，文件名不直观
- **建议**: (1) 导出前弹输入框让用户填"项目名/日期" (2) 增加"一键全部导出"按钮

### M7. 无新手引导/操作教程
- **现象**: 首次进入看到 7 个面板 + 3D 场景 + 底部 AI 对话，无任何引导
- **影响**: 新用户不知道可以"点裂缝看详情""点告警飞到位置""拖滑块调阈值"
- **建议**: 首次访问弹出 3-4 步引导浮层（类似 shepherd.js），可跳过，localStorage 记忆

### M8. AI 设置无连接测试
- **现象**: SettingsDialog 填入 API Key 后只能点保存，无法测试是否连通
- **影响**: 用户填了一个错误的 Key，保存后切换到对话面板发消息才发现不工作
- **建议**: 加"测试连接"按钮，发送一个简单 ping 请求验证

### M9. 字体极小 — 长时间使用疲劳
- **现象**: 大量使用 `text-[8px]` `text-[9px]` `text-[10px]`。机器人卡片的信号强度、时间戳只有 8px
- **影响**: 在控制室 4K 大屏上这些文字不可读；矿长老花眼根本看不清
- **建议**: (1) 最小字号不低于 10px (2) 提供字号缩放控件（小/中/大）

### M10. 物理真实模式水印过于激进
- **现象**: 启用后整个 3D 场景蒙上 30% 黑色遮罩 + 巨大旋转水印文字"未经 AI 润色"
- **影响**: 测绘工程师启用此模式是为了看原始数据，但水印遮挡了视线，反而看不到数据
- **建议**: 水印缩小到右下角小印章；或仅保留斜线条纹背景，移除大字

## 三、中度体验问题 (MEDIUM — 有改进空间)

### ME1. 无告警声音开关偏好
- 没有告警声音设置。应提供"告警音量"和"免打扰时段"配置

### ME2. 机器人列表无排序
- 可以按状态/型号/Mesh角色筛选，但不能按电量、信号强度、最近更新排序
- **建议**: 列表头部加排序按钮（电量↓/信号↓/时间↓）

### ME3. 裂缝节点点击后无独立传感器详情
- 点击测点会飞到3D位置，但右侧面板仍显示裂缝级别数据，看不到该节点独立传感器的完整19项读数
- **建议**: 选中节点后切换到"节点详情视图"，展示该节点独立读数

### ME4. 标注不跨场景保留
- 切换数据源（如煤矿→管线）时，所有剖面线/文字标注被清除，无法恢复
- **建议**: 标注按场景独立保存，切换回来时恢复

### ME5. 合规声明条不可关闭
- 底部红色合规声明永久占据 40px 高度。用户已经看过 1000 次了
- **建议**: 提供"已阅"折叠按钮，折叠后缩小为一个小图标

### ME6. 无裂缝/区域搜索
- 机器人列表有搜索框，但裂缝网络没有。在大型场景中找特定裂缝只能手动点击
- **建议**: 右侧面板顶部加搜索框

### ME7. 传感器趋势窗口不可调
- 固定显示"近 2.5 小时"，无法切换 1h/6h/24h
- **建议**: 加时间范围选择器

### ME8. 无全屏模式
- 3D 场景和 AI 对话不能独立全屏。演示时受限
- **建议**: 各面板加全屏按钮

### ME9. 对话无消息搜索/编辑/删除
- AI 对话历史无法搜索关键词、不能编辑已发送消息、不能删除消息

### ME10. 流式输出突然重排
- 流式过程中渲染纯文本，完成后突然切换到 Markdown 格式，导致内容跳动
- **建议**: 流式过程中也渲染基本 Markdown（至少加粗/列表），避免突变

### ME11. 无加载骨架屏
- 各面板加载时只显示"加载中..."文字。骨架屏更专业

### ME12. 导出无历史记录
- 导出后无记录。无法追溯"上次给安监局交的是哪份报告"

### ME13. GasThresholdSlider 对管线/核反应堆场景标签混乱
- 滑块标签只适配了 coal/gold/oil，对 pipeline/nuclear/refinery/underground 显示的标签/单位可能不正确

### ME14. ExportHub 关闭后状态不重置
- 导出完成后 done 状态保留。关闭再打开仍显示上次的"已导出"提示，可能误导用户

### ME15. 无时区/UTC 显示
- 仅显示本地时间。对跨国矿企需要 UTC 或可配置时区

## 四、轻度体验问题 (MINOR — 锦上添花)

### MI1. 版本号 "v1.0" 是占位符
- Header 中 `v1.0` 不是真实版本号，应从 `package.json` 读取

### MI2. 机器人搜索无结果时无操作建议
- 显示"无匹配机器人"但不建议"清除筛选条件"

### MI3. 无面包屑导航
- 深入裂缝节点后无返回路径指示

### MI4. 图层开关无记忆
- 切换场景后图层状态重置为默认。应记住用户偏好

### MI5. 对话快捷指令不可自定义
- 用户不能添加自己的常用指令

### MI6. 颜色体系小不一致
- 部分组件仍用硬编码色值而非 `sceneColors.ts`（如 `#44AAFF` vs `#4DA6FF`）

### MI7. 无暗/亮模式切换
- 控制室长时间暗色模式阅读疲劳

### MI8. 无数据对比模式
- 无法同时对比两个时段/两个区域的数据
